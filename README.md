# FitTrack - UNER 

## Descripción

FitTrack UNER es una aplicación móvil que permitirá a los usuarios registrar sus caminatas o entrenamientos al aire libre, visualizando la ruta recorrida en un mapa, tomando fotos durante la actividad y recibiendo recordatorios para ejercitarse.

### Pantallas creadas:

1. **Inicio (Home / index.jsx)**  
   - Pantalla principal del drawer.  
   - Contiene un botón para abrir el menú lateral.  
   - Muestra información general y acceso rápido a otras pantallas.

2. **Perfil de Usuario (profile.jsx)**  
   - Pantalla de perfil donde se pueden visualizar y editar datos del usuario.  

3. **Configuración (settings.jsx)**  
   - Pantalla para ajustar preferencias de la app.  

4. **Autenticación**  
   - **Login**: formulario para iniciar sesión.  
   - **Register**: formulario para registrar nuevos usuarios.  
   - Validación de campos usando **Formik y Yup**.
